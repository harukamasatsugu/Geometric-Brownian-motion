# Geometric-Brownian-motion
幾何ブラウン運動を日経平均株価に当てはめてみた

幾何ブラウン運動は、株価の動きをモデル化する確率過程の一つである。株価の動きは長期的には右肩上がりであるものの、短期的にはランダムな変化が見られる。

𝑑𝑆(𝑡)=𝛼𝑆(𝑡)𝑑𝑡+𝜎𝑆(𝑡)𝑑𝑊(𝑡)

𝑆(𝑡): 時刻tにおける株価
𝛼: 期待収益率（ドリフト）
𝜎: 株価のボラティリティ（ボラティリティ）
𝑑𝑊(𝑡): ブラウン運動（ランダムな変動）

今回は、日経平均株価のヒストリカルデータを用いて、𝛼 と𝜎　の数値を求めてみました。
